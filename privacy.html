Kebijakan Privasi

Terakhir diperbarui: Juni 2025

1. Ringkasan

ambilin adalah web downloader video gratis yang berjalan langsung di browser kamu. Kami berkomitmen untuk melindungi privasi kamu. Kebijakan ini menjelaskan data apa yang kami kumpulkan, bagaimana kami menggunakannya, dan hak kamu.

2. Data yang Kami Kumpulkan

Kami TIDAK menyimpan video yang kamu unduh. Semua proses download berjalan langsung di browser kamu. Link video yang kamu tempel dikirim ke:

  • tikwm.com (API publik TikTok downloader) untuk mengambil metadata video

Server kami tidak menyimpan log permanen dari URL yang diproses.

3. Penyimpanan Lokal (LocalStorage)

Kami menggunakan localStorage browser kamu untuk menyimpan:

  • Preferensi tema (dark/light mode) — agar tetap konsisten saat refresh
  • Riwayat 5 download terakhir — untuk memudahkan re-download. Data ini HANYA ada di browser kamu, nggak dikirim ke server mana pun

Kamu bisa menghapus riwayat kapan saja lewat tombol "Hapus" di section Riwayat, atau dengan clear browser data.

4. Cookie

ambilin tidak menggunakan cookie untuk tracking. Kami hanya menggunakan localStorage seperti dijelaskan di atas.

5. Google Analytics

Kami menggunakan Google Analytics 4 (GA4) untuk memahami bagaimana pengunjung menggunakan web ini, agar kami bisa meningkatkan pengalaman pengguna. Data yang dikumpulkan:

  • Halaman yang dikunjungi (anonim)
  • Event interaction: klik download, install, share, theme toggle, dll
  • Performa web (Web Vitals: LCP, FID, CLS)
  • Negara asal (agregat, bukan IP individual)

Kami mengaktifkan anonymize_ip sehingga IP kamu tidak disimpan oleh Google. Kamu bisa opt-out pakai add-on Google Analytics Opt-out.

6. Layanan Pihak Ketiga

Saat download, web kami mengakses API pihak ketiga:

  • tikwm.com — API publik TikTok downloader
  • quickchart.io — generate QR code untuk fitur transfer HP
  • Google Fonts — font Inter & Poppins

Masing-masing layanan ini punya kebijakan privasi sendiri. Kami nggak bertanggung jawab atas data yang dikirim ke mereka.

7. Hak Kamu

Sebagai pengguna, kamu punya hak untuk:

  • Mengakses data yang tersimpan di browser kamu (lewat DevTools → Application → LocalStorage)
  • Menghapus data lokal kapan saja (tombol "Hapus" di Riwayat, atau clear browser data)
  • Opt-out dari analytics (pakai add-on Google Analytics Opt-out)
  • Memakai web ini tanpa JavaScript (dengan fungsi terbatas)

8. Keamanan

Web ini di-hosting di GitHub Pages (HTTPS) dan Cloudflare Workers (HTTPS). Semua komunikasi dienkripsi. Namun, ingat bahwa download video dari API pihak ketiga bisa saja diekspos di log server mereka — kami nggak kontrol itu.

9. Anak di Bawah Umur

ambilin tidak ditujukan untuk anak di bawah 13 tahun. Kami nggak sengaja mengumpulkan data dari anak-anak. Kalau kamu orang tua dan menemukan anak kamu menggunakan web ini tanpa supervisi, mohon bantu clear browser data mereka.

10. Perubahan Kebijakan

Kebijakan ini bisa berubah sewaktu-waktu. Versi terbaru selalu ada di halaman ini dengan tanggal update di atas. Lanjut menggunakan web ini setelah perubahan = kamu setuju dengan kebijakan baru.

Punya pertanyaan soal privasi?
Buka issue di GitHub repository kami, atau lihat juga halaman DMCA untuk masalah hak cipta.
