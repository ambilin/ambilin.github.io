Generate PNG Icons

Klik kedua tombol di bawah untuk mengunduh icon-192.png dan icon-512.png. Letakkan kedua file tersebut di folder yang sama dengan file ambilin kamu (index.html, manifest.json, dll).

Setelah kedua PNG ada di folder, tombol Install di ambilin akan langsung berfungsi di Chrome Android. Manifest.json sudah dikonfigurasi untuk membaca kedua file ini otomatis.
