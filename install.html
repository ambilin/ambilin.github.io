Cara Install ambilin

Install ambilin sebagai aplikasi di perangkat kamu — buka lebih cepat, bisa terima link langsung dari TikTok, dan jalan seperti aplikasi native. Pilih platform di bawah untuk panduan lengkap.

Android iPhone / iPad Desktop

Android

  1. Buka Google Chrome di HP kamu (bukan browser bawaan seperti Samsung Internet).

  2. Kunjungi ambilin.github.io — tunggu sampai halaman muncul sempurna.

  3. Tekan tombol Install (ikon lingkaran dengan panah) di pojok kanan atas navbar.

  4. Atau alternatif: buka menu ⋮ di pojok kanan atas Chrome → pilih "Tambah ke Layar utama" atau "Install aplikasi".

  5. Kotak dialog konfirmasi muncul → tekan Install.

  6. Tunggu beberapa detik — selesai! Logo ambilin muncul di layar utama HP kamu. 🎉

💡 Tips: Kalau tombol Install tidak muncul, pastikan kamu pakai Chrome (versi terbaru). Beberapa HP Samsung mungkin hanya menampilkan opsi lewat menu ⋮.

iPhone / iPad

  1. Buka Safari — wajib Safari, bukan Chrome iOS. Apple hanya mengizinkan install PWA dari Safari.

  2. Kunjungi ambilin.github.io — tunggu sampai halaman muncul sempurna.

  3. Tekan tombol Share di bawah layar (ikon kotak dengan panah ke atas ↑).

  4. Scroll ke bawah di daftar yang muncul → cari dan pilih "Tambah ke Layar Utama".

  5. Konfirmasi nama aplikasi (bisa diubah) → tekan Tambah di pojok kanan atas.

  6. Selesai! Logo ambilin muncul di layar utama. Saat dibuka, ambilin jalan fullscreen tanpa address bar — seperti aplikasi native. 🎉

💡 Tips: Kalau opsi "Tambah ke Layar Utama" tidak ada, scroll lebih ke bawah di menu Share — kadang tersembunyi. Atau edit menu Share dengan tekan "Edit" di paling bawah daftar.

Desktop (PC / Laptop)

  1. Buka Google Chrome atau Microsoft Edge di komputer kamu.

  2. Kunjungi ambilin.github.io.

  3. Cari ikon ⊕ (Install) di sebelah kanan address bar, di ujung URL.

  4. Atau alternatif: tekan tombol Install di pojok kanan atas halaman ambilin.

  5. Klik Install di dialog yang muncul.

  6. Selesai! ambilin terbuka sebagai aplikasi terpisah, dengan ikon di taskbar (Windows) atau Dock (Mac). Tidak ada tab browser — jalan seperti aplikasi biasa. 🎉

💡 Tips: Aplikasi yang sudah terinstall bisa di-search di Start Menu (Windows) atau Spotlight (Mac). Untuk uninstall: klik-kanan ikon → Uninstall.

Share langsung dari TikTok

Setelah ambilin terinstall, kamu bisa share video langsung dari aplikasi TikTok — tanpa perlu copy-paste link manual:

  1. Buka aplikasi TikTok di HP kamu.

  2. Pilih video yang mau diunduh.

  3. Tekan tombol Share (panah ke kanan →) di sisi kanan video.

  4. Di daftar aplikasi yang muncul, cari dan pilih ambilin.

  5. ambilin terbuka otomatis, link terisi, dan langsung diproses — tinggal pilih kualitas & download!

💡 Catatan: Fitur share-target ini hanya jalan setelah ambilin di-install sebagai aplikasi (PWA). Kalau belum install, kamu tetap bisa copy link manual dan tempel di ambilin.
